Platypus.js

The best way to create your web and desktop apps

Platypus.js web appplications

To assemble and run web Platypus.js application with Gradle, use the following commands:

                                
git clone https://github.com/marat-gainullin/platypus-gradle-skeleton.git
cd platypus-gradle-skeleton
gradlew runWebApp

Platypus.js Java SE appplications

To run two tier application:

Clone git repository of Skeleton project, using the following commands:

                                
git clone https://github.com/marat-gainullin/platypus-gradle-skeleton.git
cd platypus-gradle-skeleton

Then edit ./src/private.properties file. Add the following properties there:

                                
run.serverType=none
run.clientType=platypus

And then launch the application, using the following command:

                                
gradlew platypusClient

To run client server application:

Clone git repository of Skeleton project, using the following commands:

                                
git clone https://github.com/marat-gainullin/platypus-gradle-skeleton.git
cd platypus-gradle-skeleton

Then edit ./src/private.properties file. Add the following properties there:

                                
run.serverType=platypus
run.clientType=platypus

Then launch Platypus.js server, using the following command:

                                
gradlew platypusServer

And then launch Platypus.js client, using the following command:

                                
gradlew platypusClient

Source code

Platypus.js is hosted on GitHub.

Requirements

To run Platypus.js applications you need JRE 8 update 121 or higher.
Apache Tomcat 8.5.8 or higher is required for running web applications.
