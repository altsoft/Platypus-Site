Notes

If you have problems with platypus.js updating process on Windows ("file not found" message or nothing happens after clicking on update message in platypus designer), you may fix this by using one of folowing approaches:

  • Replace C:\ProgramData\Oracle\Java\javapath\ in Windows System PATH variable with "C:\Program Files\Java\jre1.8.0_60\bin"
  • Replace softlink like C:\ProgramData\Oracle\Java\javapath\javaw.exe by hardlink using command mklink /H <link> <target> (mklink /H C:\ProgramData\Oracle\Java\javapath\javaw.exe "C:\Program Files\Java\jre1.8.0_60\bin\javaw.exe").
  • Wait until Microsoft will fix symlink problems, or Oracle will stop using them in java's installer.
